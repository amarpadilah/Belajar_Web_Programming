Apa Itu CSS Selector?

CSS Selector adalah sesuatu yang digunakan pada CSS untuk mengenali sebuah elemen HTML yang akan diberi style.

Ada beberapa cara untuk menggunakan selector di CSS, antara lain :

  • elemen HTML
  • id
  • class
  • complex selector
