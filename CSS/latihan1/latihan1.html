Apa itu CSS?

CSS (Cascade Style Sheet) adalah sebuah bahasa untuk mengatur tampilan web sehingga terlihat lebih menarik dan indah. Dengan CSS, kita dapat mengatur layout (tata letak), warna, font, garis, background, animasi, dan lain-lain.


Ada 3 cara untuk menggunakan CSS, yaitu :

  1. Internal CSS

    2. Internal CSS adalah CSS yang langsung ditulis di dalam file HTML nya dengan tag 'style{}'. Penempatannya ada didalam tag "head" dibawah tag "title".

  2. External CSS

    3. External CSS adalah CSS yang penempatannya diluar file HTML nya yakni punya file sendiri dengan extensi '.css'.

  3. Inline CSS

    4. Inline CSS adalah CSS yang langsung digunakan pada tag atau atribut tertentu didalam file HTML nya
