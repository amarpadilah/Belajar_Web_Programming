Text Styling

Dalam text styling ada beberapa properti, antara lain yaitu :

  • color

    • Untuk memberi warna pada tulisan

  • text-align

    • Untuk mengatur format paragraf atau teks

  • text-indent

    • Untuk memberi identasi pada paragraf

  • text-decoration

    • Untuk mengatur koneksi pada teks

  • text-transform

    • Untuk megubah jenis huruf menjadi huruf besar, kecil, atau kapital

  • letter-spacing

    • Untuk mengatur spasi antar huruf

  • word-spacing

    • Untuk mengatur spasi antar kata
