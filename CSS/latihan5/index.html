CSS 5 Background

Properti pada Background ada 4, yaitu :

  • background-color

    • Untuk mengatur warna pada background

  • background-image

    • Untuk mengatur gambar yang akan digunakan pada background

  • background-position

    • Untuk mengatur posisi gambar pada background

  • background-repeat

    • Untuk mengatur jenis pengulangan gambar pada background
