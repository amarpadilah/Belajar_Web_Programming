CSS 8 Pseudo Class

apa itu Pseudo Class?

Kelas semu yang dimiliki oleh sebuah elemen HTML, yang membuat kita dapat mengdefinisikan style pada "keadaan tertentu" dari element tersebut.

Pseudo Class yang berhubungan dengan link, antara lain :

  • :link

    style default pada sebuah link (a yang memiliki href)

  • :hover

    style ketika kursor mouse berada diatas link/element apapun

  • :active

    style ketika sebuah link di-klik (keadaan aktif)

  • :visited

    styke ketika sebuah link sudah pernah dikunjungi sebelumnya (menggunakan browser yang sama)

Pseudo Class yang berhubungan langsung dengan elemen, antara lain :

  • :first-child

    memilih elemen pertama dari sebuah parent (elemen pembungkusnya)

  • :last-child

    memilih elemen terakhir dari sebuah parent (elemen pembungkusnya)

  • nth-child('isi parametrnya')

    memilih elemen ke-n dari sebuah parent(elemen pembungkusnya). n bisa berarti urutan (1), (2),.... atau pola (2n), (3n+2), (4n-1) atau ganjil dan genap (even) & (ood).

Pseudo Class yang berhubungan dengan posisi elemen, antara lain :

  • first-of-type

    memilih elemen pertama dari sebuah jenis atau tipe tag

  • last-of-type

    memilih elemen terakhir dari sebuah jenis atau tipe tag
