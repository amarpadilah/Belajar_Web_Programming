font styling

ini adalah cara untuk menambahkan font pada html, ini adalah beberapa jenis atribut css untuk font :

  • font-family

    • untuk mengatur jenis font yang digunakan

  • font-size

    • untuk mengatur ukuran font yang digunakan(biasanya dalam ukuran px, %, em.)

  • font-weight

    • untuk mengatur ketebalan font yang digunakan

  • font-variant

    • untuk mengubah font menjadi small caps

  • font-style

    • untuk mengubah font menjadi bercetak miring

  • line-height

    • untuk mengatur spasi antara baris
